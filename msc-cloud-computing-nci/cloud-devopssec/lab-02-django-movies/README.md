**Cloud Platform Programming – NCI**

This section presents the practical work developed for the *Cloud Platform Programming* module of the Master’s Degree in Cloud Computing at the National College of Ireland (NCI).

The module focuses on hands-on learning, enabling students to work with cloud infrastructure, virtual machines, secure environments, and web application development using Python-based frameworks.

---

## Module Overview

- Introduction to cloud-based development environments  
- Use of virtual environments for Python projects  
- Web application development with Django  
- Database integration and migrations  
- Model–View–Template (MVT) architecture  
- Django Admin interface for data management  

---

## Key Skills Developed

- Python web application development  
- Use of Django framework  
- Virtual environment management  
- Database modeling and migrations  
- CRUD operations using Django Admin  
- URL routing, views, and templates  
- Deployment and testing using AWS Cloud9  

---

## Practical Labs

### **Lab 02: Django Web Application Development**

This lab focuses on building a functional web application using the Django framework.

The following tasks were completed:

- Creation of a Django project and application  
- Configuration of a Python virtual environment  
- Definition of database models and execution of migrations  
- Registration of models in the Django Admin interface  
- Implementation of views and URL routing  
- Use of templates to render dynamic content  
- Management of movie records (Create, Read, Update, Delete)  
- Execution and testing of the application using the Django development server  

The application includes:
- A **Movies** application  
- An **Admin interface** to manage movie records  
- A **public web page** displaying the list of movies  
- A **detail page** for individual movie entries
