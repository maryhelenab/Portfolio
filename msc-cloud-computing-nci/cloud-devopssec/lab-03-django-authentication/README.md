# Cloud Platform Programming – NCI

This lab is part of the **Cloud Platform Programming** module of the **Master’s Degree in Cloud Computing** at the **National College of Ireland (NCI)**.

The objective of this lab is to implement **user authentication and authorization** in a Django web application using Django’s built-in authentication system.

---

# Lab 03 – Django Authentication

This lab focuses on adding authentication features to a Django application, allowing users to register, log in, log out, and access protected content.

Django provides a powerful built-in authentication system that handles user management, password hashing, sessions, and access control.

---

# Key Concepts

* Django authentication framework
* User registration and login
* Session management
* Access control using authentication decorators
* Password hashing and security
* User management through Django Admin

---

# Tasks Completed

The following tasks were implemented during the lab:

* Creation of a Django application for authentication
* Configuration of Django authentication system
* Implementation of user registration functionality
* Implementation of login and logout features
* Creation of templates for authentication pages
* Protection of specific views for authenticated users
* Configuration of URL routing for authentication views

---

# Application Features

The application includes the following functionality:

* **User Registration Page** – allows users to create a new account
* **Login Page** – allows users to authenticate using username and password
* **Logout Functionality** – securely ends the user session
* **Protected Views** – pages that can only be accessed by authenticated users
* **Admin Panel** – user management via Django Admin interface
