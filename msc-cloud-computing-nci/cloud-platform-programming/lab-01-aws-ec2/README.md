**Lab 01 – AWS EC2 Fundamentals**

This lab introduces the fundamental concepts of Amazon Elastic Compute Cloud (EC2) as part of the Cloud Platform Programming module of the Master’s Degree in Cloud Computing at the National College of Ireland (NCI).

The objective of this lab is to provide hands-on experience in launching, accessing, and managing virtual machines in the AWS cloud environment.

---

**Lab Objectives**

- Launch an EC2 instance using the AWS Management Console  
- Configure key pairs and security groups  
- Understand instance states and lifecycle  
- Connect to a Linux instance using SSH and PuTTY  

---

**Key Skills Developed**

- Cloud infrastructure provisioning  
- Secure access to virtual machines  
- Basic Linux instance management  
- Understanding of AWS EC2 core components  

---

**Technologies Used**

- Amazon Web Services (AWS)
- EC2 (Elastic Compute Cloud)
- Amazon Linux / Ubuntu
- SSH & PuTTY
