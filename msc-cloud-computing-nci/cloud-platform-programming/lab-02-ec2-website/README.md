## Project Description
This project demonstrates the deployment of a static website on AWS EC2 using Nginx and GitHub for version control.

## Steps
1. Launched an Ubuntu EC2 instance
2. Configured Security Group for SSH and HTTP
3. Installed and started Nginx web server
4. Generated SSH keys and connected EC2 to GitHub
5. Cloned GitHub Portfolio repository to EC2
6. Deployed website files to /var/www/html

## Technologies
- AWS EC2 (Ubuntu 22.04)
- Nginx Web Server
- Git & GitHub
- HTML5
