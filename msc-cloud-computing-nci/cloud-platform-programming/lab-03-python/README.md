**Lab 03 – Python Basics**

This lab introduces fundamental Python programming concepts within the context of the Cloud Platform Programming module of the Master’s Degree in Cloud Computing at the National College of Ireland (NCI).

The purpose of this lab is to develop basic scripting skills that support automation and problem-solving in cloud computing environments.

---

**Lab Objectives**

- Understand Python syntax and core concepts  
- Write and execute basic Python scripts  
- Apply programming logic to simple problems  
- Prepare for more advanced cloud automation tasks  

---

**Key Skills Developed**

- Python programming fundamentals  
- Logical thinking and problem-solving  
- Script-based automation foundations  
- Code structuring and readability  

---

**Technologies Used**

- Python 3
- Linux environment
- Command-line tools
