**Lab 04 – Flask Dynamic Routes and Forms**

This lab focuses on building a basic web application using Flask as part of the *Cloud Platform Programming* module of the Master’s Degree in Cloud Computing at the National College of Ireland (NCI).

The objective of this lab is to introduce backend web development concepts, including routing, dynamic URLs, template rendering, and handling user input through HTML forms.

---

**Lab Objectives**

* Understand Flask application structure
* Implement static and dynamic routes
* Create reusable HTML templates
* Process form input using GET and POST methods
* Pass data from backend to frontend

---

**Key Concepts Practiced**

* Dynamic routing with URL parameters
* Template rendering using Jinja2
* Form handling with request methods
* Variable passing between Python and HTML
* Debugging runtime and template errors

---

**Technologies Used**

* Python 3
* Flask
* HTML
* Jinja2
* VS Code
* Command Line Terminal

---

**Learning Outcome**

By completing this lab, the student gained practical experience in building a functional Flask web application, understanding backend–frontend interaction, and debugging common development errors such as routing issues, template loading problems, and syntax mistakes.
